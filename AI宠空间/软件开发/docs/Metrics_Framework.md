# PetCare AI 宠护平台 - 核心指标体系 (Metrics Framework)

## 1. 北极星指标 (North Star Metric)
- **指标名称**: **月度成功履约订单量 (Monthly Completed Orders, MCO)**
- **定义**: 每个月由平台接收并由门店成功完成履约（状态变更为“已完成”）的有效订单总数。
- **业务意义**: 衡量平台核心业务增长、用户需求匹配度及门店履约能力的综合指标。

## 2. HEART 模型指标体系

| 维度 (Dimension) | 关键指标 (Key Metrics) | 衡量标准 (Measurement) |
| :--- | :--- | :--- |
| **幸福度 (Happiness)** | 用户评分 (NPS) | 用户完成预约后的满意度评分 (1-5 星)。 |
| | 系统报错率 | 提交预约时触发 Error 状态的百分比。 |
| **参与度 (Engagement)** | 平均下单频率 | 单个活跃用户在 30 天内的平均下单次数。 |
| | 页面留存时间 (下单流) | 用户从打开 `home.html` 到提交订单的平均时长 (越短越好)。 |
| **采用度 (Adoption)** | 新用户转化率 | 首次访问并完成首单预约的用户占比。 |
| | 手机验证码登录成功率 | 登录环节的流失率分析。 |
| **留存度 (Retention)** | 30 天复购率 | 30 天内有第二次下单行为的用户占比。 |
| | 活跃用户次月留存 | 上月活跃用户在下月继续访问的比例。 |
| **任务完成度 (Task Success)** | 预约成功率 | 成功创建订单数 / 尝试提交次数。 |
| | 平均派单耗时 | 订单从“待派单”转为“处理中”的平均分钟数。 |

## 3. AARRR 模型 (增长漏斗)
- **获取 (Acquisition)**: 杭州试点门店自然引流、微信裂变转化。
- **激活 (Activation)**: 完成首次宠物档案录入与下单。
- **留存 (Retention)**: 通过高质量履约与增值服务复购。
- **收入 (Revenue)**: **客单价 (ARPU)**、**增值服务渗透率 (Add-on Attachment Rate)**。
- **推荐 (Referral)**: 分享“最近订单”截图或裂变链接。

## 4. 经营分析指标 (Admin Board)
- **预算偏差率 (Budget Variance Rate)**: 实际支出与预估收入的偏离程度。
- **门店饱和度 (Store Utilization)**: 门店当前处理中订单数 / 门店最大承载力。
- **调度人效 (Dispatch Efficiency)**: 单个管理员日均处理分派订单数。

## 5. 监控与告警方案
- **实时看板**: `admin.html` 仪表盘每 1 分钟自动刷新一次核心 KPI。
- **阈值告警**:
  - 当“待派单”订单超过 15 分钟未处理，触发红色高亮预警。
  - 当“系统报错率”超过 5% 时，自动触发技术团队告警。
- **多智能体预警 (Future)**: 分析 Agent 自动识别异常波动的根因并推送建议。
